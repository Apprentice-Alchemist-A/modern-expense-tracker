## レスポンシブ対応

### モバイル（～768px）
- サイドバーは引き出し式
- カード表示を優先
- タブナビゲーション

### タブレット（768px～1024px）
- サイドバー固定、幅調整
- リスト・カード表示の切り替え

### デスクトップ（1024px～）
- 全機能表示
- マルチカラム表示対応
